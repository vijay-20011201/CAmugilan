What is Import Export Code (IEC)?

Importer Exporter Code (IEC) is a ten-digit unique code issued by the Director-General of Foreign Trade (DGFT), Ministry of Commerce, Government of India.

IEC is compulsory for any import or export in India. Without IEC, no person or entity can legally import/export goods (except for personal use not connected with trade, manufacturing, or agriculture).

We at MKDA assist entities in obtaining new IEC registration, amendments/modifications to existing registrations, and more.

Importance of IEC

Import Export Code (IEC) or Import Export License is mandatory for every exporter/importer to conduct legitimate import-export business in India.

IEC Registration Procedure

IEC registration is completely online and requires a Digital Signature Certificate (DSC).

  1. Obtain Class 3 Digital Signature Certificate (DSC)
  2. Fill the online application form accurately and upload required documents
  3. Submit the application and follow up for issuance if delayed

Documents Required

For Individuals / Proprietorship

  • PAN Card
  • Aadhaar Card / Voter ID / Passport / Driving License
  • Cancelled Cheque of Current Account
  • Passport Size Photo

For Company / LLP / Partnership

  • PAN Card of Establishment
  • PAN Card of Authorized Person
  • Aadhaar / Voter ID / Passport / DL of Authorized Person
  • Passport Size Photo
  • Cancelled Cheque of Current Account

Frequently Asked Questions

What happens if an IEC application is rejected?
If the DGFT has rejected your e-IEC application, then the system will allow you to either view your rejection letter or reapply.
Whether the Import Export Code (IEC) is mandatory for the import of goods?
IEC is not required for the import or export of goods for personal use but is mandatory only for commercial purposes and business
